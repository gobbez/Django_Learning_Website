Che cos'è Django?

Django è un framework web open-source scritto in Python. È progettato per facilitare lo sviluppo rapido di applicazioni web sicure e mantenibili. Django segue il principio del "batterie incluse", fornendo tutti gli strumenti necessari per costruire applicazioni web moderne senza dover reinventare la ruota.

Alcune caratteristiche principali di Django includono:

  • ORM potente: Una mappatura oggetto-relazionale che permette di lavorare con database utilizzando oggetti Python.
  • Sistema di template: Un motore di template flessibile per generare dinamicamente contenuti HTML.
  • Sistema di routing URL: Una configurazione intuitiva per mappare URL alle viste.
  • Sicurezza integrata: Protezioni incorporate contro vulnerabilità comuni come SQL injection e XSS.

Un esempio semplice di vista in Django

from django.http import HttpResponse

def hello_world(request):
    return HttpResponse("Ciao, mondo!")

Questa funzione semplice restituisce una risposta HTTP con il testo "Ciao, mondo!". Questo è un esempio basilare di una vista in Django.
