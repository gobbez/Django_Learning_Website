Setup e Installazione di Django

Prima di iniziare con Django, dobbiamo assicurarci che Python e pip siano installati sul nostro sistema. Django richiede Python 3.8 o superiore.

Verifica dell'installazione di Python

Apri il terminale e digita:

python --version
# o
python3 --version

Se non Ã¨ installato, scarica Python dal sito ufficiale python.org.

Installazione di Django

Usa pip per installare Django:

pip install django
# o per l'installazione globale
pip3 install django

Verifica l'installazione:

django-admin --version

Ora sei pronto per creare il tuo primo progetto Django!
