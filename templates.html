Masterizzare i Templates in Django

I templates in Django sono il modo per separare la presentazione del documento dalla logica di business. Un template contiene la struttura statica di un documento HTML insieme ai tag speciali che descrivono come il documento verr√† popolato dinamicamente.

Django ha il suo sistema di template con caratteristiche potenti per il riutilizzo del codice.

Variabili

Le variabili vengono circondate da {{ e }}. Esempio: {{ variable }}

Tags

I tag forniscono logica nei templates. Alcuni comuni: {% if %}, {% for %}, {% block %}

Filtri

I filtri cambiano come una variabile viene visualizzata. Esempio: {{ value|date:"D d M Y" }}
