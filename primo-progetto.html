Creazione del Primo Progetto Django

Ora che Django è installato, creiamo il nostro primo progetto. Un progetto Django è una collezione di configurazioni per un'istanza di Django, inclusi database e opzioni.

Creare un nuovo progetto

django-admin startproject myproject
cd myproject

Questo crea una struttura di directory per il progetto.

Avviare il server di sviluppo

python manage.py runserver

Visita http://127.0.0.1:8000/ nel browser per vedere la pagina di benvenuto di Django.

Creare una nuova app

Django utilizza il concetto di app. Creiamo una app chiamata 'blog':

python manage.py startapp blog

Ora abbiamo la base del nostro progetto Django!
